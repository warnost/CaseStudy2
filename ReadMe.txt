This Repository has files related to an attrition analysis for Frito Lay  

William_Arnost_Case2.html and William_Arnost_Case2.Rmd contain the full analysis code.  
CaseStudy2-data.csv is the primary data file.  
Case2PredictionsArnost Attrition.csv contains my Attrition model predictions
CaseStudy2CompSet No Attrition.csv is the input for the final predictions  
Case2PredictionsArnost Salary.csv contains my Salary model predictions  
CaseStudy2CompSet No Salary.csv is the input for the final predictions 

Frito Lay Attrition Analysis.pptx is the final powerpoint presentation
